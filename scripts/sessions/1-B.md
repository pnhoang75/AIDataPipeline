### SourceConnector ABC (connector.py)
```python
from abc import ABC, abstractmethod
from typing import Iterator
from events import RawDocumentEvent

class SourceConnector(ABC):
    @abstractmethod
    def poll(self) -> Iterator[RawDocumentEvent]: ...

    @abstractmethod
    def ack(self, event_id: str) -> None: ...
```

### S3Connector requirements
- Use `minio` Python client (not boto3) for MinIO compatibility
- Watermark stored as Redis HSET `connector:{connector_id}:watermark` → `last_seen` ISO timestamp
- On each poll: list objects modified after watermark, publish RawDocumentEvent to Kafka `raw-documents`
- If Kafka produce fails: retry with exponential backoff (100ms × 2^n, cap 30s, max 5 attempts)
- After 5 failures: emit `connector_errors_total{reason="kafka_timeout"}` counter, skip file, DO NOT advance watermark
- Filter by `file_types` allowlist from config
- Write `source_file_status` row with status='pending' before publishing to Kafka

### RawDocumentEvent schema (events.py)
Fields: event_id (uuid4), source_type ("s3"), source_id (bucket/key), content_ref ("s3://bucket/key"),
        content_type (MIME), metadata (dict), ingested_at (unix timestamp), tenant_id (str)
